# 霍格沃茨关卡设计 v4：方向校正版

## 方向基准

这版不再把霍格沃茨做成普通棋盘，而是按常见霍格沃茨校地地图的空间关系来设计：

- 北向在画面上方，地图左上角放北箭头。
- `魁地奇球场` 位于城堡外侧偏西北。
- `黑湖` 位于城堡南侧 / 西南侧，城堡靠湖岸建立。
- `船屋` 位于黑湖和城堡之间的湖岸。
- `禁林` 位于校地东侧 / 东北侧。
- `海格小屋` 位于禁林边缘、城堡外侧。
- `温室`、`打人柳` 放在城堡与禁林之间的校地区域。

这个方向不是复制某一张具体商业地图，而是采用影视、游戏和常见校地示意图中最稳定的关系：城堡临黑湖，禁林在外缘，魁地奇场在城堡外侧，海格小屋靠近禁林。

## 拓扑设计

主线是从城堡核心向外控场：

`霍格沃茨城堡 -> 活动楼梯 -> 礼堂 -> 船屋 -> 黑湖`

西北外围线：

`格兰芬多塔 -> 北草坪 -> 魁地奇球场`

东侧禁林线：

`霍格沃茨城堡 -> 打人柳 -> 海格小屋 -> 禁林`

南侧地牢线：

`活动楼梯 -> 礼堂 -> 斯莱特林地牢`

节奏线：

`活动楼梯 -> 霍格沃茨城堡 -> 有求必应屋 -> 温室 -> 禁林`

## 为什么这版方向更合理

这版把“真实校地感”放在第一位：

- 黑湖不是右侧小块，而是南侧/西南侧的大水域。
- 船屋贴在湖岸，不再漂在普通灰色领地边缘。
- 城堡在湖边高地，而不是被禁林和黑湖夹成横向格子。
- 禁林在外缘东侧，海格小屋位于城堡和禁林之间。
- 魁地奇球场放到西北外围，符合“离城堡有距离的运动场”感觉。

## 资源与任务节奏

每关开局金币为 `50`。玩家每次向非己方地块发起抢占行动时，消耗 `15` 金币；向己方地块调兵不消耗金币。这个规则用于限制无脑连续扩张，迫使玩家在抢地、买法术牌、守关键点之间取舍。

任务从单目标改为三步任务链。每完成一步会给金币奖励，并推进到下一步目标，例如第一关为 `夺回活动楼梯 -> 控制礼堂补给线 -> 清除地牢威胁`。

## 法术牌

法术牌只属于玩家。敌方 AI 不会购买或释放咒语，只会按兵力、路线、特殊地块价值进行普通出兵。

| 法术牌 | 费用 | 作用 |
| --- | ---: | --- |
| 增援令 | 40 | 己方目标地块 +30 兵力 |
| 守护神咒 | 45 | 己方目标地块获得 10 秒护盾 |
| 飞路粉 | 35 | 从一个己方地块快速调兵到另一个己方地块 |
| 统统石化 | 35 | 敌方或中立地块 7 秒内不能增长和出兵 |
| 除你武器 | 30 | 敌方地块 9 秒内出兵数量减半 |

## 随机事件

随机事件会在战斗中隔一段时间触发，让同一关的节奏出现变化：

- `凤凰福克斯出现`：最薄弱的己方地块 +18 兵力。
- `摄魂怪来袭`：全场兵力增长短时间变慢。
- `活动楼梯移动`：活动楼梯临时连通温室和有求必应屋。
- `发现学院补给`：金币 +20。

## 特殊地块机制

### 有求必应屋

首次占领 `有求必应屋` 时，拥有者兵力最少的两个己方区域各获得 +12 兵力。它位于城堡内部偏东上层，不单独放在校地外侧。

### 魁地奇球场

第一个占领 `魁地奇球场` 的阵营获得 +100 兵力，直接加到球场本地，用来制造一次强力反扑或滚雪球。

### 三把扫帚

我方占领 `三把扫帚` 后持续产金币，金币速度明显高于普通地块。这个点位在霍格莫德村北侧，适合中后段补道具。

### 天文塔

我方占领 `天文塔` 后，敌方下一次进攻会先显示预警方向，并在地图上画出红色虚线。玩家可以提前撤兵、防守或反打。

### 石像防线

占领方在 `石像防线` 受到的伤害降低。它放在礼堂、地牢和船屋之间，是南线防守核心。

### 黑湖

`黑湖` 初始兵力高、增长慢。从黑湖发出的部队移动稍慢。

### 打人柳

`打人柳` 是右路阻滞点，适合用来挡禁林反扑并连接有求必应屋、温室和禁林。

## AI 行为

- 敌方开局从 `斯莱特林地牢` 和 `禁林` 两端制造压力。
- 如果玩家贪 `魁地奇球场`，敌方优先抢 `活动楼梯`。
- 如果玩家抢 `黑湖`，敌方从 `温室 / 打人柳` 切入城堡东侧。
- 终局敌方优先保 `禁林`，然后反推 `海格小屋`。
