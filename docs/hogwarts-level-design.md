# 霍格沃茨关卡设计 v4：方向校正版

## 方向基准

这版不再把霍格沃茨做成普通棋盘，而是按常见霍格沃茨校地地图的空间关系来设计：

- 北向在画面上方，地图左上角放北箭头。
- `魁地奇球场` 位于城堡外侧偏西北。
- `黑湖` 位于城堡南侧 / 西南侧，城堡靠湖岸建立。
- `船屋` 位于黑湖和城堡之间的湖岸。
- `禁林` 位于校地东侧 / 东北侧。
- `海格小屋` 位于禁林边缘、城堡外侧。
- `温室`、`打人柳` 放在城堡与禁林之间的校地区域。

这个方向不是复制某一张具体商业地图，而是采用影视、游戏和常见校地示意图中最稳定的关系：城堡临黑湖，禁林在外缘，魁地奇场在城堡外侧，海格小屋靠近禁林。

## 拓扑设计

主线是从城堡核心向外控场：

`霍格沃茨城堡 -> 活动楼梯 -> 礼堂 -> 船屋 -> 黑湖`

西北外围线：

`格兰芬多塔 -> 北草坪 -> 魁地奇球场`

东侧禁林线：

`霍格沃茨城堡 -> 打人柳 -> 海格小屋 -> 禁林`

南侧地牢线：

`活动楼梯 -> 礼堂 -> 斯莱特林地牢`

节奏线：

`活动楼梯 -> 霍格沃茨城堡 -> 有求必应屋 -> 温室 -> 禁林`

## 为什么这版方向更合理

这版把“真实校地感”放在第一位：

- 黑湖不是右侧小块，而是南侧/西南侧的大水域。
- 船屋贴在湖岸，不再漂在普通灰色领地边缘。
- 城堡在湖边高地，而不是被禁林和黑湖夹成横向格子。
- 禁林在外缘东侧，海格小屋位于城堡和禁林之间。
- 魁地奇球场放到西北外围，符合“离城堡有距离的运动场”感觉。

## 特殊机制

### 活动楼梯

占领 `活动楼梯` 后，每 10 秒开放一次短暂捷径，可向 `霍格沃茨城堡` 或 `温室` 快速调兵。

### 有求必应屋

首次占领 `有求必应屋` 时，拥有者兵力最少的两个己方区域各获得 +12 兵力。它位于城堡内部偏东上层，不单独放在校地外侧。

### 黑湖

`黑湖` 初始兵力高、增长慢。从黑湖发出的部队移动稍慢，但抵达损耗更低。

### 禁林

`禁林` 每 12 秒向 `海格小屋` 或 `温室` 施压。玩家拖得越久，东侧越容易崩。

### 打人柳

`打人柳` 是右路阻滞点。占领方获得短暂防守优势，适合用来挡禁林反扑。

## AI 行为

- 敌方开局从 `斯莱特林地牢` 和 `禁林` 两端制造压力。
- 如果玩家贪 `魁地奇球场`，敌方优先抢 `活动楼梯`。
- 如果玩家抢 `黑湖`，敌方从 `温室 / 打人柳` 切入城堡东侧。
- 终局敌方优先保 `禁林`，然后反推 `海格小屋`。
